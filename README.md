# Employee Management System

A Python + SQLite project demonstrating database management concepts.

## Features

* Create Database and Tables
* CRUD Operations (Create, Read, Update, Delete)
* Primary Key
* Foreign Key
* WHERE Clause
* ORDER BY Clause
* COUNT() Function
* INNER JOIN

## Technologies Used

* Python
* SQLite
* VS Code
* Git & GitHub

## Project Files

* table.py – Creates database tables
* insert.py – Inserts employee records
* read.py – Displays employee records
* update.py – Updates employee details
* delete.py – Deletes employee records
* where.py – Demonstrates WHERE clause
* orderby.py – Demonstrates ORDER BY clause
* count.py – Demonstrates COUNT() function
* join.py – Demonstrates INNER JOIN operation

## Learning Outcomes

This project demonstrates the implementation of SQL operations using Python and SQLite, including database creation, table relationships, CRUD operations, filtering, sorting, aggregation, and joins.
